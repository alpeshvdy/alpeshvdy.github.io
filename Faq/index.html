Over the time there are some changes occurred in the checklist, and items on checklist always increased. Meanwhile AHT target had a reduction of a minute. CheckList AuTo Opener helps to reduce effort and save time of agents. It opens "almost all" tabs with required google searches and social media sites also with appropriate searches where possible. It may also help agents to stay organized and help agents to stay on track. Since it opens most of the required tabs probably agent won't forget any checklist item.
Agents require to paste data in formats given in page "ForMats!", then click the appropriate button according to region of the user (i.e USA, Canada, Mexico, Australia). In site will remove the unnecessary things from pasted data and also convert it to required format as per the checklist. Finally, it will open checklist tabs.
This site does not store any kind of data. After execution of any button it will reset the input field as well as all the variables of the javascript attached to the buttons. To substantiate my claim you can review the source code in my github repository at https://github.com/alpeshvdy/alpeshvdy.github.io and there is no other contributor or any other github user authorized to make changes in the code.
